NAME

symlink_list - Show the file to which a symlink points

SYNOPSIS

symlink list [-name] <file name> [-help]

DESCRIPTION

The symlink list command shows if a file is a symbolic link and, if so, shows to which file the link points.

CAUTIONS

The symlink commands are only available on Windows platforms. On other platforms, use the readlink command.

OPTIONS

[-name] <file name>

Specify one or more symbolic links to list. The -name is optional, but the file name is not.

-help

Prints the online help for this command. All other valid options are ignored.

OUTPUT

This command prints "'file' is a symlink to 'otherfile'" or "'file' is not a symlink."

EXAMPLES

Given that file mylink is a link pointing to file myfile, the following commands will list the files:

   % symlink list -name mylink
   'mylink' is a symlink to 'myfile'

   % symlink list mylink
   'mylink' is a symlink to 'myfile'

   % symlink list -name myfile
   'myfile' is not a symlink.

   % symlink list myfile
   'myfile' is not a symlink.

PRIVILEGE REQUIRED

The issuer must have the l (look) and r (read) permissions on the ACL of the directory that houses the symlink.

SEE ALSO

symlink(1), symlink_make(1), symlink_remove(1)
